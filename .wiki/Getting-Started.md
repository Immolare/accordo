# Getting Started

Accordo works in any modern browser — nothing to install.

## 1. Open the tuner

Go to **[accordo.tools](https://accordo.tools)** and tap **Start**. Your browser will ask for microphone access: allow it so Accordo can hear your instrument.

> 💡 No microphone? You can still tune by ear with **Sound mode** (see below).

## 2. Pick your instrument and tuning

- Choose **Guitar** or **Bass** at the top.
- Tap the tuning selector (e.g. *Standard E · 6 strings*) to browse 50+ tunings. You can search by name (`drop c`, `open g`) or by notes.
- Don't see yours? [Create a custom tuning](Custom-Tunings) and share it with a link.
- Use the **♭ Flat** button to shift any tuning down a half step instantly.

## 3. Tune each string

1. Play a string. With **Auto** enabled, Accordo detects which string you're playing.
2. Watch the needle:
   - **Left (♭)** — too low, tighten the string ↑
   - **Right (♯)** — too high, loosen the string ↓
   - **Green center** — in tune!
3. Hold the note in the green zone for a second: the string is validated with a ✓ (and a vibration on mobile).
4. When all strings are validated, you're done 🎉

## Tuning by ear (Sound mode)

Tap **🔊 Sound**, then tap any string to hear a realistic reference tone. Match your string to the sound — this works even without microphone access.

## Calibration

The reference pitch defaults to **A4 = 440 Hz**. Use the **− / +** buttons in the top bar to adjust it from 415 to 465 Hz (e.g. 432 Hz, baroque 415 Hz).

## Install as an app

Accordo is a PWA:

- **Android / Chrome**: menu → *Add to Home screen*
- **iOS / Safari**: Share → *Add to Home Screen*

Once installed it works **fully offline**.

## Tips for a clean reading

- Tune in a quiet room, close to the microphone.
- Play one string at a time and let it ring.
- Pluck near the 12th fret for a purer tone with fewer harmonics.
