# Supported Tunings

All tunings available in Accordo, exactly as they appear in the app. Notes are listed from the lowest string to the highest. The **♭ Flat** switch shifts any of them down one half step.

## Guitar — 6 strings

### Standards

| Tuning | Notes |
|---|---|
| Standard E | E A D G B E |
| Standard Eb | Eb Ab Db Gb Bb Eb |
| Standard D | D G C F A D |
| Standard C# | C# F# B E G# C# |
| Standard C | C F Bb Eb G C |
| Standard B | B E A D F# B |

### Drops

| Tuning | Notes |
|---|---|
| Drop D | D A D G B E |
| Drop C# | C# G# C# F# A# D# |
| Drop C | C G C F A D |
| Drop B | B F# B E G# C# |
| Drop A# | A# F A# D# G C |
| Drop A | A E A D F# B |

### Opens

| Tuning | Notes |
